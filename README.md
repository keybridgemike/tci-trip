# tci-trip-planner

Live site: https://keybridgemike.github.io/tci-trip/

## View locally

- Dev server (default): `npm run dev` then open `http://localhost:5173/`
- Dev server (alt port): `npm run dev:15176` then open `http://localhost:15176/`
- Production build: `npm run build`
- Preview build: `npm run preview` (or `npm run preview:15176`)

Stop the server with `Ctrl+C`.
